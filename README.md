# coin_flutter

## Getting Started

## Dependencies

* mockito [link](https://pub.dev/packages/mockito)
* build_runner [link](https://pub.dev/packages/build_runner)

## Commands

Run tests:

```shell
flutter test
```

Build runner:

```shell
dart run build_runner build
```

Build runner (watch):

```shell
dart run build_runner watch
```

Get dependencies:

```shell
flutter pub get
```
